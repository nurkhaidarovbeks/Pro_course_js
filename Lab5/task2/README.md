# Lab 5.2 README

## Summary
This task implements a modular JavaScript application using static and dynamic imports.

## Modules
- app.mjs
- core.mjs
- utils.mjs
- constants.mjs
- advanced_feature.mjs
- lazy_component.mjs

## Features
- Static imports for core modules
- Dynamic import on interaction
- Dynamic import on visibility
- Error handling for lazy-loaded modules

## Run
Open index.html in a browser and test:
1. Button click for advanced feature
2. Scroll down for lazy component loading
