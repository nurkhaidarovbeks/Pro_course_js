# Lab 4.1 README

This task focused on identifying anti-patterns in legacy JavaScript code.

## Identified Anti-Patterns
- Global namespace pollution
- Tight coupling
- Code duplication
- String-based setTimeout() / setInterval()
- Spaghetti code
- Inline JavaScript / DOM-coupled logic

## Files
- anti_patterns_identified.md
- anti_patterns_classification.md
- impact_assessment.md

## Conclusion
The legacy code contains multiple anti-patterns that negatively affect maintainability, scalability, and team productivity. Refactoring with Module Pattern and Namespace Pattern is recommended.
