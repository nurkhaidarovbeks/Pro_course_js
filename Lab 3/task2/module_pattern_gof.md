# Module Pattern (GoF Style) 

## Pattern Name
Module Pattern 

## Description 
Encapsulates private variables and exposes a public API.

## Context 
Used to organize code and prevent global namespace pollution.

## Problem 
Global variables can conflict and cause bugs.

## Solution 
Use closures and return an object with public methods.

## Benefits 
- encapsulation 
- better code structure 

## Trade-offs
- slightly more complex code
